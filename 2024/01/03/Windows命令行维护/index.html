Windows命令行维护

发表于2024/01/03|更新于2024/02/04
|阅读量:

文章不仅针对新大陆物联网职业技能大赛，如果有其他需要记录的，文章会持续更新

通过 netstat 查询 端口是否被打开。

netstat -ano | findstr : 80
netstat -ano 显示所有端口、以数字形式显示所有ip和端口、显示占用的进程的ID
findstr：80 寻找筛选”:80”字符串的内容，并显示。

请将使用命令窗口查询 TCP/IP 数据包从工作站电脑到云平台服务器连接

tracert ndp.nlecloud.com

tracert 寻找跳板 +地址/域名

清除DNS缓存

ipconfig /flushdns

flush:冲洗

生成目录树到指定文件

tree [路径] > [指定文件]

eg: tree D:\竞赛目录 > A-15-1.txt

复制并重命名

当前目录下：copy [源文件] [目标目录]\[目标文件名]

eg :copy text.txt c:\doc\key_bak.txt

寻找文件内容并输出

查找字样，找到了显示“成功”，没找到显示“不成功”。

find "目标字样" [路径\目标文件] && echo "成功" || echo"不成功"

eg：find "ok" c:\doc\key_bak.txt && echo "成功" || echo "不成功"

仅查看初始安装日期和系统启动时间

systeminfo |findstr -i "初始安装日期 系统启动时间"
